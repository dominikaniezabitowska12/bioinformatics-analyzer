# 🧬 BioAnalyzer

BioAnalyzer is a desktop bioinformatics application written in Python using Tkinter.

The application allows users to perform basic DNA and protein sequence analyses through a simple graphical interface.

---

## Features

### 🧬 DNA Sequence Analysis

- DNA sequence validation
- Sequence length calculation
- Nucleotide composition
- GC content calculation
- Start codon detection
- Stop codon detection
- Coding sequence verification

### 🧪 Protein Analysis

- DNA to protein translation
- Amino acid sequence generation
- Protein molecular weight calculation
- Premature STOP codon detection

### 🧬 Mutation Analysis

- Comparison of two DNA sequences
- Detection of:
  - substitution
  - insertion
  - deletion

---

## Technologies

- Python
- Tkinter

---

## Project Structure

- GUI created with Tkinter
- Modular code divided into analysis modules
- Separate windows for each type of analysis
